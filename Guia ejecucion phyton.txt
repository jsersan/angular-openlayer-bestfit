1. Copia el script anterior en un nuevo archivo en tu proyecto, por ejemplo: verifica_ciclos.py

2. Asegúrate de tener los tres archivos en el mismo directorio:

asignacion.ts
Oferta-por-familias-profesionales.pdf
institutos.ts
verifica_ciclos.py

3. Instala PyPDF2 si no lo tienes:

bash
pip install PyPDF2

4. Ejecuta el script:

    bash
    python verifica_ciclos.py

El script mostrará en consola las discrepancias (si existieran) entre ciclos y centros de los tres archivos. 
Si está todo bien, verás la conformidad ciclo por ciclo.
